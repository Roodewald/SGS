# ASP.NET Тестовое Задание

В этом тестовом задании я решил продемонстрировать свои навыки работы с ASP.NET и паттернами проектирования.

Я считаю, что лучше было бы использовать следующие маршруты в этом тестовом задании:

- `GET /currencies` - возвращает коллекцию валют.
- `GET /currencies/${name}` - возвращает цену выбранной валюты.

## Документация

CurrencyDataFetcher - класс, в который внедряются 2 зависимости:

- IHttpClientFactory - используется для реализации устойчивых HTTP-запросов.
- IMemoryCache - позволяет сохранять данные в кэше на сервере.

## API Справка

### Получить список валют

```http
GET /currencies
```

| Parameter    | Type     | Description                              |
| :---------   | :------- | :--------------------------------------- |
| `pageSize`   | `int32 ` | Сколько элементов отображать на странице |
| `pageNumber` | `int32 ` | Номер страницы                           |

#### Получить значение валюты

```http
  GET /currency/${name}
```

| Parameter | Type     | Description                                          |
| :-------- | :------- | :--------------------------------------------------- |
| `name `   | `string` | **Обязательный**. Идентификатор валюты для получения |



## Запуск локально

Клонировать проект

```bash
  git clone https://github.com/Roodewald/SGS
```

Перейти в директорию проекта:

```bash
  cd SGS
```

Установить зависимости:

```bash
  Target SDK: net8
```

Запустить приложение:

```bash
  dotnet run
```
